# implement-of-NTRU
ANTS‘98经典版 NTRU公钥加密算法的C++实现

求多项式模逆算法使用效率秦九韶大衍求一术实现。

为提高速度，本程序采用较弱安全性参数，使用者可自行更改更安全的参数

# 计划实现的功能

1.对文件的加解密

2.本代码采用ECB（电子密码本加密模式），计划实现安全性更高的CBC加密模式

3.当前加密后的密文字符串较长，计划对密文字符串压缩处理  (已实现）
